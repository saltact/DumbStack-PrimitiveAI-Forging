## 🛠️ How I Take Notes

All notes in these directories are written in standard Markdown (`.md`). 

Mathematical formulas and equations are rendered using LaTeX to ensure precision and readability. For example, my notes heavily feature structures like this:

$$A \mathbf{x} = \lambda \mathbf{x}$$
*(Eigenvalue Equation)*

Where applicable, theoretical math concepts are immediately paired with practical C++ or Python (NumPy) scripts to prove the methodology in code.
